The Code Breakers — Facens (landing page)

Como usar:
1. Substitua as imagens de placeholder em /assets/ pelos arquivos reais (ex.: assets/pedro.jpg). Mantenha os nomes ou atualize o src correspondente no index.html.
   Exemplo de comentário no código: <!-- Coloque a foto aqui: assets/pedro.jpg -->

2. Substitua cada URL do LinkedIn diretamente no index.html:
   - href="COLE_AQUI_O_LINKEDIN_DO_PEDRO"
   - href="COLE_AQUI_O_LINKEDIN_DO_LEONARDO"
   - href="COLE_AQUI_O_LINKEDIN_DA_LUANA"
   - href="COLE_AQUI_O_LINKEDIN_DA_YASMIN"
   - href="COLE_AQUI_O_LINKEDIN_DO_EDUARDO"
   - href="COLE_AQUI_O_LINKEDIN_DO_PEDRO_PAULO"

3. Abra o arquivo index.html no navegador (duplo clique ou "Abrir com" -> navegador) para visualizar a landing page localmente.
